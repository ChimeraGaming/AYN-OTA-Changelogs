# AYN-OTA-Changelogs

This repository contains a community-maintained archive of AYN OTA changelogs.

It is intended for users who do not use Discord or who prefer a clean, chronological record of OTA changes without announcements, discussions, or unrelated messages.

Currently tracking OTA updates for AYN Thor. Additional devices may be added in the future.

---

## Contributing

Contributions are welcome.

If you would like to help, please ensure that:
- Entries reflect official OTA changelog information only
- No speculation, commentary, or announcements are included
- Dates, version numbers, and wording are preserved as accurately as possible

Small corrections, formatting fixes, and new OTA entries are all appreciated.
