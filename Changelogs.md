# AYN Thor — Software Changelog

## 2026
<details>
<summary>View 2026 updates</summary>

### Thor v1.0.0.360 (2026-01-13)
<details>
<summary>View changes</summary>

- Fixed an issue in the Bilibili app where sliding on the screen to adjust brightness would incorrectly adjust the brightness of both displays.
- Made the smart fan speed more aggressive to address overheating issues in high-performance mode.
- Added customizable fan speed curves in Smart Fan Mode.
- Fixed an issue where third-party Android launcher icons were not displayed on the lower screen (requires opening via the search bar).
- Fixed an issue where the maximum custom fan speed was limited to 70 percent in TCC and Game Assistant.
- Fixed washed-out icon and wallpaper colors on the upper screen after exiting HDR video playback or HDR streaming.
- Updated Wi-Fi firmware to resolve connection failures and system freeze or reboot issues.
- Updated audio EQ configuration to improve speaker bass performance.
- Added OLED screen protection features, including dynamic anti-image retention and burn-in protection.
- Added support for 80 percent charge limit, 5V2A, and 9V2A charge limits when battery State of Health is below 80 percent.

</details>

## 2025

### Thor v1.0.0.293 (2025-11-20)
<details>
<summary>View changes</summary>

- Fixed broken app switcher and task manager when using alternative launchers.
- Added support to pin the on-screen keyboard to the bottom screen.
- Added support for DisplayPort output when docked.
- Fixed system freeze during the first screen mirroring attempt after factory reset on Wi-Fi 7 networks.
- Added option to hide Android navigation bar from the bottom screen.
- Enabled maximum fan speed in custom fan mode.
- Fixed random line artifacts on screen edges when launching JScreenFix.
- Added boot animation.
- Added option to disable the power button while the device is closed.
- Prevented system wake on charging cable plug-in when the lid is closed.
- Added support for using the AYN key to wake the screen.

</details>

- Updated to Linux mainline kernel v6.17.5.

</details>

### Thor v1.0.0.279 (2025-10-20)
<details>
<summary>View changes</summary>

- Fixed system crash when setting Android stock wallpaper with an image below 1240×1080 resolution.
- Fixed TCC clock displaying only 12-hour format regardless of system setting.
- Fixed inability to unlock the device when lock screen password was enabled and TCC was set to bottom-screen-only mode.
- Fixed bottom screen lighting up unexpectedly and TCC becoming unresponsive when using vertical screen apps in top-screen-only mode.
- Added support for setting wallpapers independently on the bottom screen.
- Fixed occasional wake and sleep failures when rapidly opening or closing the lid.

</details>
